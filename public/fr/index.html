Ressources communauté Ekylibre

Gérez toutes les données de votre exploitation en un seul endroit avec
le système d'information agricole open source de référence.

Essayer la démo (FR|EN) Lire la documentation utilisateur Rejoindre Discord (FR|EN)

Opérations obligatoires

Ekylibre intègre de nombreuses opérations obligatoires : interventions culturales, comptabilité, ventes, achats, et bien d'autres.

La durabilité d'abord

Ekylibre inscrit son développement dans une doctrine de durabilité afin de préserver l'activité et la vie agricoles.

Visualisations puissantes

Ekylibre embarque de nombreuses bibliothèques pour analyser les données, avec des tableaux de bord contextuels et un système de cellules personnalisable.

Modèle complet

Ekylibre fournit un système de base de données solide basé sur PostgreSQL / PostGIS, avec plus de 300 modèles pour stocker une grande diversité de données.

Données sécurisées

Chaque exploitation Ekylibre s'exécute sur son propre tenant pour offrir le maximum de sécurité à vos données.

Référentiels de données

Ekylibre fournit les systèmes de nomenclatures Onoma et Lexicon pour normaliser les données. Ces systèmes contiennent plus de 20 millions de références.

Personnalisation

Ajoutez des champs sur chaque modèle selon vos besoins. Si ce n'est pas suffisant, des bibliothèques de plugins vous permettent d'adapter le code à vos propres usages.

Intégrations

Les intégrations existantes permettent de connecter des données tierces à Ekylibre : RTK GPS, données animales, météo, capteurs de sol et bien d'autres.

Merci à nos contributeurs :

Open Source

Ekylibre est 100 % open source et porté par la communauté. Tous les composants sont disponibles sur GitHub.
